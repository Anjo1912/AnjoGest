🚀 Anjo Gestão - Dashboard & Cobrança
Este projeto foi evoluído para incluir ferramentas de Business Intelligence (BI) voltadas para prestadores de serviço.

✨ Novidades desta versão:
Dashboard Financeira: Monitoramento de metas mensais (R$ 5.000,00) com barra de progresso dinâmica.

Gráficos de Performance: Comparativo visual dos últimos 3 meses com normalização de dados e linha de alvo (Target Line).

Geração de PDF: Sistema de cobrança automatizado que gera relatórios de serviços pendentes por loja e permite compartilhamento instantâneo via WhatsApp.

Foco em UX: Interface simplificada para agilizar o dia a dia do faturamento.
